# WealthWise — Private Local Ledger

WealthWise is a modern, privacy-first, and highly secured personal financial ledger. It is designed to empower individuals with total ownership over their financial data by executing **100% client-side operations** with cryptographically secure, browser-isolated storage. No servers, no remote trackers, and zero external exposure.

---

## 🚀 Key Functional Modules

- **📊 Interactive Charts (Recharts)**: High-resolution category breakdown pie charts and monthly budget utilization bar graphs.
- **🎯 Dynamic Budgeting Control**: Set custom limits across various categories or configure a global monthly budget. Visual indicators dynamically warn you of overspending.
- **💡 Smart Financial Insights**: Automatically calculates spend rates, tracks transaction counts, and delivers actionable alerts regarding your current pacing and potential savings.
- **💼 Action Deck Ledger**: An intuitive and responsive table interface supporting full search, category filtering, and real-time ledger item creation/editing/deletion.
- **✨ Instant Preview Access**: Explore the sandbox immediately using **Guest Preview** or **Demo Account** quick-access toggles, pre-seeded with beautiful, customizable records.

## 🛠️ Technology Stack

- **Framework**: [React 19](https://react.dev/) + [TypeScript](https://www.typescriptlang.org/) (Type-safe compilation)

- **Data Visualization**: [Recharts](https://recharts.org/) (Smooth animative charts)

## 💻 Getting Started

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. Clone or download this project.
2. Navigate to the project root directory and install dependencies:
   ```bash
   npm install
   ```

### Development Server

To boot up the local high-speed Vite development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your preferred browser.

### Build Compilation

To compile a production-ready, highly optimized bundle inside the `dist/` folder:

```bash
npm run build
```
